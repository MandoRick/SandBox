Copy the unzipped theme folder to the extensions folder you created.

If the Arduino IDE is running, select File > Quit from the Arduino IDE menus to exit all windows.
Start the Arduino IDE.
Select File > Preferences from the Arduino IDE menus.
Open the "Theme" menu.
Select the name of the new theme from the menu.
